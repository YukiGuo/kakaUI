### API
| 属性          | 说明           | 值  |
| ------------- |:-------------:| -----:|
| value        |输入框 默认值    |  String |
| disabled    |是否禁用状态，默认为false     | Boolean |
| error   |错误信息   |  String |
| success   |表示成功输入状态   |   Boolean |
| v-model   |   绑定的值   |    任意值|
