# Popover 气泡卡片

### 示例

<ClientOnly>
<popover-demo></popover-demo>
</ClientOnly>

### API
| 属性          | 说明           | 值  |
| ------------- |:-------------:| :-----|
| position  |气泡位置, 默认为top  |   `top`, `bottom`, `left`, `right`|
| trigger   |触发方式，默认为click触发    | `click` `hover` |
