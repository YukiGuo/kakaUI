### API
| 属性          | 说明           | 值  |
| ------------- |:-------------| :-----|
| selectedName  |默认展开的Item项数组 |  k-collapse-item的name组成的数组|
| single  |只允许一个item展开，默认为true    | `true `false` |
| name|必需：k-collapse-item 的name|   `String` |
| title  |必需：k-collapse-item的title   | `String` |
