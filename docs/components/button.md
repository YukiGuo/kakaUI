# Button 按钮

### 示例
<ClientOnly>
<button-demo></button-demo>
</ClientOnly>

### API

| 属性          | 说明           | 值  |
| ------------- |:-------------| :-----|
| icon-name  | 图标名 |`left` `right` `setting` `success` `loading` |
| icon-position  | Icon位置默认在左  |`left` `right` |
| loading    |设置按钮载入状态,默认为false     |  `false` `true` |
