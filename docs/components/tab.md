### API

| 属性          | 说明           | 值  |
| ------------- |:-------------| :-----|
| selectedTab |必需： 默认选中的标签页 | name值 |
| direction  | tab的方向，默认为横向 |`horizontal` `vertical` |
| name   |必需：k-tab-item 和k-tab-pane组件 的名字     |  `String` |
